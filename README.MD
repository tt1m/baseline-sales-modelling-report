# Basline Sales Modelling Report 

The report is built using [Zensical](https://zensical.org/), a Python-based documentation generator.

---

## 🚀 Getting Started

To view this documentation locally, ensure you have Python installed and Zensical set up in your environment.

### Prerequisites
* [Python 3.10+](https://www.python.org/)
* `pip` (Python package manager)

### View Report Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/tt1m/baseline-sales-modelling-report.git
   cd baseline-sales-modelling-report
   ```

2. (Recommended) Create and activate a virtual environment:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
   ```

3. Install Zensical:
   ```bash
   pip install zensical
   ```

4. Serve the documentation:
   ```bash
   zensical serve
   ```

5. Open your browser to the local address provided (typically `http://localhost:8000`).
